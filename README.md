# 🏏 IPL Data Analysis

A personal data analysis project where I explored IPL cricket data 
to find interesting patterns and insights using Python.

## 🤔 Why I Built This
I wanted to practice data analysis with a dataset I actually enjoy — cricket! 
This is my first EDA project as part of my Data Science learning journey.

## 📊 What I Analyzed
- Which team won the most matches?
- Does winning the toss help win the match?
- Who are the top run scorers and wicket takers?
- Which venues favor which teams?
- Who hits the most sixes and fours?
- Best catch takers (excluding wicket keepers)
- Centuries by Indian players

## 🛠️ Tools Used
Python, Pandas, Matplotlib, Seaborn, Jupyter Notebook

## 📁 Dataset
Ball by ball IPL match data — matches.csv & deliveries.csv

## 🚀 How to Run
```bash
git clone https://github.com/Sarathi08sk/IPL-Data-Analysis.git
pip3 install pandas numpy matplotlib seaborn jupyter
jupyter notebook
```
